# RUTAYISIRE SERGE BLAISE - CV Portfolio

A modern, dark-themed CV portfolio website built with React and Tailwind CSS.

## 🚀 Features

- 📱 Responsive design
- 🌙 Dark theme
- 📄 Multi-page layout (Home, Projects, Profiles)
- 🔗 Social media integration (LinkedIn, Instagram, Facebook, GitHub)
- 📁 Project showcase with GitHub links
- 🖼️ Profile image support

## 🛠️ Technologies

- React 18
- React Router DOM
- Tailwind CSS (via CDN)
- CSS Custom Properties
